Sobre a Uvateca

A Uvateca é um projeto de biblioteca digital simulada, criado como parte de um portfólio de desenvolvimento web. O objetivo é demonstrar habilidades em HTML, CSS e JavaScript para criar uma aplicação web funcional e moderna.

Nesta plataforma, clientes podem navegar pelo acervo, enquanto funcionários e administradores possuem painéis dedicados para gerenciar o conteúdo.

Quem Contatar?

Administração do Site:
Para questões sobre gerenciamento de usuários, use o e-mail: admin@uvateca.com.br

Suporte (Funcionários):
Para problemas com o cadastro de livros, use o e-mail: fnc@uvateca.com.br
